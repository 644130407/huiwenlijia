<h1>说明</h1>
<div>该项目采用了Node.js并配合Nunjucks模板引擎进行开发</div>
<div>UI设计采用了amazeui框架，实现了移动优势，兼容PC，可以在不同设备上实现不同的显示效果</div>
<div>系统要求：采用node.js v7.6极其以上版本</div>
<div>安装完成后，需要运行安装模块依赖</div>
